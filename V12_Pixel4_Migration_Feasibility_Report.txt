Pixel 4 Manual Test Instructions
Run these in Termux from the QuantCore repo root.

1. Check Python:
python --version

2. Install dependencies:
python -m pip install --upgrade pip
python -m pip install -r requirements.txt

3. Import dependency check:
python -c "import pandas,numpy,ccxt,aiohttp; print('dependency import PASS')"

4. Confirm Binance testnet secrets are present:
python -c "import os; print(bool(os.getenv('BINANCE_TESTNET_KEY')), bool(os.getenv('BINANCE_TESTNET_SECRET')))"

5. Binance futures testnet connectivity:
curl -I "https://testnet.binancefuture.com/fapi/v1/exchangeInfo"

6. ccxt Binance futures testnet OHLCV smoke check:
python - <<'PY'
import asyncio
import ccxt.async_support as ccxt

async def main():
    ex = ccxt.binance({"enableRateLimit": True, "options": {"defaultType": "future"}})
    ex.set_sandbox_mode(True)
    ex.urls["api"]["public"] = "https://testnet.binancefuture.com"
    ex.urls["api"]["private"] = "https://testnet.binancefuture.com"
    try:
        rows = await ex.fetch_ohlcv("BTC/USDT:USDT", "1h", limit=2)
        print(rows)
    finally:
        await ex.close()

asyncio.run(main())
PY

7. V12 no-order scan:
python scripts/run_v12_scan_once.py
